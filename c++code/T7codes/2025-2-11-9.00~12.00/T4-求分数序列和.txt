# 求分数序列和
### 时间限制：C/C++ 1000MS，其他语言 2000MS
### 内存限制：C/C++ 256MB，其他语言 512MB
### 分数：100
##  描述
有一个分数序列q1/p1，q2/p2，q3/p3，q4/p4，q5/p5，…  
其中q(i+1)=qi+pi，p(i+1)=qi，p1=1，q1=2。  
比如这个序列前几项为2/1，3/2，5/3，8/5，13/8，21/13…  
输入两个正整数m、n，求这个分数序列第m 到 第n项之和，结果保留三位小数输出
## 输出描述
## 用例输入 1 
```
4
6
```
## 用例输出 1
``` 
4.840
```
## 用例输入 2 
```
1
16
```
用例输出 2 
```
26.188
```
