# 正常血压:
### 时间限制：C/C++ 1000MS，其他语言 2000MS  
### 内存限制：C/C++ 64MB，其他语言 128MB  
### 分数：100  
## 描述  
监护室每小时测量一次病人的血压，若收缩压在 90～140 之间并且舒张压在 60～90 之间（包含端点值）则称之为正常。现给出某病人若干次测量的血压值，计算病人保持正常血压的最长小时数。
## 输入描述
第一行一个整数表示 n(n<100)。
其后有 n 行，每行 2 个正整数，分别为一次测量的收缩压和舒张压（都在 [1,200] 范围内），中间以一个空格分隔。
## 输出描述  
输出仅一行，血压连续正常的最长小时数。  
## 用例输入 1  
```
4
100 80
90 50
120 60
140 90
```
## 用例输出 1  
```   
2
```
