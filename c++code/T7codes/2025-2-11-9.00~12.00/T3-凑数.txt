# 凑数
### 时间限制：C/C++ 1000MS，其他语言 2000MS
### 内存限制：C/C++ 64MB，其他语言 128MB
### 分数：100
## 描述
现有两个正整数 x,y，试问你在 1000 以内既不是 x 的倍数，也不是 y 的倍数，且也不是 (10?x+y)的倍数的第k大正整数是多少？  
若不存在该数则输出 -1
## 输入描述  
三个整数 x,y,k(2≤x,y≤1000,1≤k≤1000)。
## 输出描述
一个正整数表示答案。
## 用例输入 1
``` 
2 3 1
```
## 用例输出 1
``` 
997
```
## 用例输入 2
``` 
5 3 912
```
## 用例输出 2
``` 
-1
```
